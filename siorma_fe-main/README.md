### ğŸ’» The Tech Powering Our Community ğŸ’»

We've chosen a modern and robust tech stack to ensure Siorma is fast, reliable, and a joy to use.

* **Next.js:** The core framework providing a powerful foundation with Server-Side Rendering (SSR) and incredible performance, making the app feel snappy and responsive.
* **Tailwind CSS:** A utility-first CSS framework that allows us to build a beautiful, custom design without leaving our HTML.
* **shadcn/ui:** We believe in stunning and accessible interfaces. Shadcn gives us a collection of elegant, composable components that are a pleasure to interact with.
* **Lucide Icons:** Crisp, clean, and consistent icons that enhance user experience and bring clarity to the interface.
* **React Context & Hooks:** For managing application state in a clean, predictable, and scalable way without the need for heavy external libraries.
* **Axios:** A reliable HTTP client for easy data fetching from API

### ğŸš€ Ready to Launch Your Own Hub? ğŸš€

Follow these simple steps to get Siorma running on your local machine.

1.  **Set Up Your HQ (Clone the Repository):**
    ```bash
    git clone https://github.com/BeLieM/siorma.git
    cd siorma
    ```

2.  **Stock the Supplies (Install Dependencies):**
    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

3.  **Get Your Campus Pass (Environment Variables):**
    Create a `.env.local` or `.env` file in the root of the project. This is where you'll put your secret keys and environment-specific configurations. You can copy the structure from `.env.example` if it exists.

4.  **Open the Gates (Run the Development Server):**
    ```bash
    npm run dev
    # or
    yarn dev
    # or
    pnpm dev
    ```
    Point your browser to the magical portal at `http://localhost:3000` and watch your local campus hub come to life!
